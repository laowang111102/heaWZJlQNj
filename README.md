# Python图书信息管理系统 python291

## 项目概述

Python图书信息管理系统是一款基于Django框架开发的Web应用程序，主要用于管理图书及其相关信息，如出版社、作者等。系统提供了一套完善的功能模块，确保用户能够高效地进行图书信息的增、删、改、查操作。

## 技术栈

- 后端：Python Django
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript

## 功能模块

- **用户模块**
  - 登录
  - 注册
- **书籍信息管理**
  - 新增书籍
  - 编辑书籍
  - 删除书籍
- **排序与搜索**
  - 书籍排序
  - 书籍搜索
- **信息管理**
  - 出版社信息管理
  - 作者信息管理
- **分页查询**

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 操作系统：兼容Windows、Linux、macOS
- Python版本：3.x
- 数据库：MySQL 5.x 或更高版本

## 部署步骤

1. 环境准备：安装Python、MySQL及必要的Python库
2. 数据库配置：创建数据库，导入初始数据
3. 程序配置：设置配置文件中的数据库连接信息
4. 运行服务：启动Django服务
